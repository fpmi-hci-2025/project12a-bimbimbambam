# Project Name
Онлайн-магазин цифровой техники 

# Description
Веб-приложение для поиска, сравнения и покупки цифровой техники с адаптивным интерфейсом для мобильных и настольных устройств. Пользователи могут фильтровать товары по характеристикам и цене, просматривать отзывы, добавлять позиции в корзину и оформлять заказ. Реализованы сценарии регистрации/входа, отслеживания статуса заказа и история покупок.
